# Neural Network Deep Learning (Incremental Learning on Large Data)

## About
This project builds a predictive neural network that supports business analytics decision-making by predicting an order quantity target using pricing and order-related inputs, with an emphasis on training scalability and model interpretability.

The workflow is designed for real-life business data and produces stakeholder-friendly artifacts (training/learning behavior, feature influence diagnostics, and interpretability plots) so outcomes can be communicated clearly beyond raw model accuracy.
