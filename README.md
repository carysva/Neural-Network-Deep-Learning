# Neural Network Deep Learning (Incremental Learning on Large Data)

## About
This project builds a predictive neural network that supports business analytics decision-making by predicting an order quantity target using pricing and order-related inputs, with an emphasis on training scalability and model interpretability.

The workflow is designed for real-life business data and produces stakeholder-friendly artifacts (training/learning behavior, feature influence diagnostics, and interpretability plots) so outcomes can be communicated clearly beyond raw model accuracy.

## Course Context (BZAN 554)
This project was completed for **BZAN 554 — Deep Learning for Business Applications (3 credit hours)**.

Course focus:
- Fundamentals of predictive analytics with business applications using deep learning techniques such as shallow and deep neural networks, convolutional neural networks, and sequence models such as recurrent and long short-term memory neural networks.
- Special focus on making deep learning work on real-life business data.

Prerequisite(s): Business Analytics 545  
Recommended background: Experience with a programming language such as Python or with deep learning  
Comment(s): Or permission of instructor.
